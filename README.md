# geojson-flow
## GeoJSON type declaration for Flow

## How to install

`npm install --save-dev geojson-flow`

## Types

 - `GeoJSONObject`:
   - `Feature`;
   - `FeatureCollection`:
   - `GeometryObject`:
    - `Point`;
    - `MultiPoint`;
    - `LineString`;
    - `MultiLineString`;
    - `Polygon`;
    - `MultiPolygon`;
    - `GeometryCollection`;

Positions are assumed to be 2-dimensional.
